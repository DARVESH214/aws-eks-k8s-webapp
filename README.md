
# 🚀 AWS EKS Kubernetes Web Application Deployment

## 📌 Overview
Deployed a containerized web application on AWS EKS using Kubernetes and exposed it to the internet.

---

## 🧠 Simple Explanation
- EKS → cloud where app runs  
- Deployment → manages pods  
- Pod → running app  
- Service → exposes app to internet  

👉 Flow:
Internet → LoadBalancer → Service → Pod → Application  

---

## ⚙️ Steps
- Created EKS cluster  
- Connected using kubectl  
- Deployed app  
- Exposed using LoadBalancer  
- Accessed via public URL  
- Scaled pods  
- Rolling updates  

---

## 🛠️ Tech Used
- AWS EKS  
- Kubernetes  
- kubectl
